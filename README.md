# Server-Monitor
Python script to monitor server uptime and local disk usage.

**Tech Used:** Python, OS, shutil  
**Features:**  
- Checks if listed servers are online (ping)  
- Displays total, used, and free disk space  

**How to run:**  
1. Run: `python monitor.py`  
2. Modify `servers` list to include servers/IPs you want to monitor
